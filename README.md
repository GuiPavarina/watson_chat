# Watson Chat

## System requirements

    Node installed v6.0.0+ 
    config.json ( private )

## Install

    npm install

## Configuration ( IMPORTANT )

    Do NOT forget to copy config.json to / folder of this project. ( Same path as app.js , package.json ...)

## Start App

    node app.js

    or

    nodemon app.js 

## Link

    Application will be available here: http://localhost:5000/
